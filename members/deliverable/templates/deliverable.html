The P2-Platypodes are proud to present our Climate Change Tracker! Please browse through our deliverable to view each team member's contributions. You can also get to know the site a little better and understand it's various features. Thanks for your visit, we hope you enjoy!

User Interactions


Home

    Home footer with various interactive elements leading to routes of many pages on the website.
    Home with general elements to describe features of the website.

Informational Pages

    Information page acts as a menu that holds the different information pages.
    Each item links to each information page
    Offers users an interactive element and choose an information page.

Quick Statistics

    Quick Statistics page displays various climate change statistics as well as some corresponding pictures.
    The pictures are presented in a gallery view for the user to interact with.


Hotspots

    Hotspots page lays out a comprehensive list of countries who are key contributors to climate change or countries that are most affected by climate change.
    Galleries and carousels are present on this page to allow user interactivity to experience the various ways certain countries have been affected.

Project Technicals


Map

    The Map page houses the visual aspect of the API.
    This page is entirely data driven and has tables that are automatically updated through iterative Jinja variables.
    The map includes an intitive key to represent certain degrees of climate change.

Login

    Login can be paired with the Register to create a credentials system for access to a dashboard
    User access to other pages in the website is not impeded.
    The entire login and register functionality is taken care of by secure SQLite databases that manage and store credentials.

Feedback

    The Feedback page is a page that allows users to give constructive comments on the website as a whole
    All data from the feedback is stored into a secure SQLite database. Collection process is entirely data driven with GET and POST.


Survey

    The Survey is a data collection tool that can allow us to gather climate change information from users and make further analysis.
    This page can be accessed throughout the website with the popup that allows you to take a quick survey for analytic purposes.
    The data is stored in a secure SQLite database and collection is entirely backend driven.

Informative Development


Causes Page

    The Causes page is a comprehensive page that details many causes of global warming.
    Every aspect of this page is organized in a professional format, similar to a newspaper article.
    The page's arrangement is creative due to it's aesthetic setup that immerses users
    The minimalist design language carries across to this page, but keeps it detailed at the same time.
    This informative page allows the users to learn something and take away some key issues that exacerbate climate change.
    Users have a better understanding of climate change and can even apply information from this page to their own lives.
